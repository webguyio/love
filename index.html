Get started

Two lines.
That's it.

1

Add the script

Paste this into your <head> or before </body>:

<script src="https://bttn.love/love.js"></script>
2

Add the button

Place this anywhere on the page. It auto-detects the current URL:

<a class="love-button"></a>

Or point it at a specific URL:

<a href="https://example.com/post/" class="love-button"></a>
3

Customize (optional)

Use any content inside the button: text, emoji, or your own SVG icon:

<a href="https://example.com/#post" class="love-button">👍</a>

Look up the count for any URL via the API:

https://bttn.love/api/love?url=https://example.com/
